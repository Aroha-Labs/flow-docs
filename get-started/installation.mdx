## 📊 Prerequisites

Before installing Mira Flows, make sure your system meets the following requirements:

### 🛠️ System Requirements

1. **Python Environment**

   - [Python 3.10](https://www.python.org/downloads/release/python-3100/) installed on your system
   - pip package manager
   - Virtual environment tool (recommended)

2. **Development Tools**
   - A code editor or IDE
   - Command-line interface access
   - Git (recommended for version control)

---

## 🔐 Creating Your Mira Flows Account

<Steps>
  <Step title="Visit the Platform">

  </Step>

## 🔄 Installing the SDK

### 🔹 Install via pip

```terminal Terminal
pip install mira-sdk
```

### 🔹 Verify Installation

```terminal Terminal
pip show mira-sdk
```

---

## 📚 Configuration

### 🔹 Initialize the Client

<CodeGroup>
```python Python
from mira_sdk import MiraClient

client = MiraClient(config={"API_KEY": "YOUR_API_KEY"})

````
</CodeGroup>

### 🔹 Environment Variables (Recommended)

1. Create a `.env` file in your project root directory:

```env .env
MIRA_API_KEY=your_api_key_here
```

2. Use it in your code:

```python Python
import os
from dotenv import load_dotenv
from mira_sdk import MiraClient

load_dotenv()
client = MiraClient(config={"API_KEY": os.getenv("MIRA_API_KEY")})
```

## ⚠ Troubleshooting

Common installation issues and their solutions:

### ❌ Python Version Mismatch

Ensure you're using Python 3.10 by checking:

```terminal Terminal
python --version
```

### 🔒 API Key Issues

Verify your API key is correctly set and has the necessary permissions.

### ⚙️ Package Conflicts

Create a fresh virtual environment if you encounter dependency issues:

```terminal Terminal
python -m venv mira-env
source mira-env/bin/activate  # On Windows: mira-env\Scripts\activate
pip install mira-sdk
```
